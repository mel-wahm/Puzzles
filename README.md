# Connect Four - 3-Player Edition

Location: `~/projects/puzzles/connect_four.py`

## Features

* **4 Discs Win Condition**: Connect **4 in a row** (horizontally, vertically, or diagonally) to claim victory.
* **Rotating Rematch Starters**: Every rematch automatically rotates who starts first (**Match 1**: Host / RED -> **Match 2**: Player 2 / YELLOW -> **Match 3**: Player 3 / GREEN -> **Match 4**: RED...).
* **3 Players**: 🔴 RED (P1), 🟡 YELLOW (P2), 🟢 GREEN (P3).
* **1 Special Move Charge Each**: 💣 `REMOVE DISC (x1)` and ⚡ `DOUBLE DROP (x1)`.
* **REMOVE DISC Improvements**: Removing an opponent's disc does not skip your turn—you drop your piece in the same turn.
* **Wi-Fi LAN & Offline Multiplayer**: Play over Wi-Fi network or locally on the same computer.

## How to Run

```bash
cd ~/projects/puzzles
python3 connect_four.py
```
